Step 1: Define Organization

Purpose

This initial step establishes the highest level of the hierarchy: the organization itself. It serves as the root container for all subsequent structures.

Functionality

  • Click the + Add Organization button to create a new entry.
  • The main screen displays a table of existing organizations with columns for ID, NAME, and DESCRIPTION.
  • You can edit or delete existing entries using the icons on the right side of the table.

Outcome

A successfully defined organization (e.g., "Akara Airlines") that acts as the foundation for the entire job architecture.
