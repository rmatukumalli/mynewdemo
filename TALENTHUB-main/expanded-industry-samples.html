Column Guide

This guide explains the purpose of each column in the job architecture framework.

Core Columns

These form the essential structure of the architecture.

  • Industry: The business sector.
  • Job Category: High-level grouping (e.g., Professional).
  • Job Function: Primary area (e.g., Technology).
  • Job Family: Specific discipline (e.g., Data Science).
  • Role Level: Defines the seniority and scope of a position within the organizational hierarchy (e.g., Junior IC, Senior IC, Manager, Executive Leader).
  • For example, a "Junior Software Engineer" (Role Level: Junior IC) vs. a "Lead Software Engineer" (Role Level: Senior IC) or a "Chief Pilot" (Role Level: Executive Leader).
  • It helps structure career progression and compensation bands.
  • Core Behavioral Anchor: The role's main purpose.

Optional / Supporting Columns

These add valuable detail but are not always essential.

  • Skill Level: Often linked to the Role Level, this broadly categorizes the required expertise for skills associated with that role (e.g., Entry-Level, Senior).
  • For instance, a "Junior Software Engineer" might require "Entry-Level" in Python, while a "Senior Software Engineer" would need "Advanced" in Python.
  • Similarly, a "First Officer" might need "Intermediate" in Aircraft Systems, while a "Captain" requires "Expert."
  • It provides a general benchmark for skill expectations at different career stages.
  • Proficiency Level: Provides granular detail on the degree of mastery for a specific skill (e.g., Beginner, Intermediate, Advanced, Expert).
  • This is crucial for precise skill gap analysis, personalized learning recommendations, and identifying subject matter experts.
  • For example, a "Pilot" might have a "Proficiency Level: Advanced" in "Boeing 737 Operations," while a "Data Analyst" might have "Proficiency Level: Intermediate" in "SQL Query Optimization."
  • Associated Skills: Specific technical or soft skills.
  • Specific Job Titles: Market-facing titles.
  • Exception / Override: Notes for special cases.
