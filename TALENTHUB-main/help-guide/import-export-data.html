Managing Job Architecture Data: Import/Export

The Import/Export Data feature allows you to manage your job architecture data by importing it from external sources (like HCM systems) and potentially exporting it for other uses.

Import Job Architecture Data

Purpose

To efficiently bring your companyâ€™s job data (e.g., from Workday, SAP SuccessFactors) into the Job Architecture Wizard by uploading a CSV file.

Functionality

  • Upload CSV File: Use the "Upload CSV File" input to select your prepared CSV file.
  • Import Data: Click the "Import Data" button to initiate the import process.
  • Import Status: A status message will appear, indicating the progress or outcome of the import.

Post-Import Data Review

After a successful import, the system provides two tabs for reviewing the processed data:

  • Data Summary: This tab provides a high-level overview, showing counts for processed categories such as:
    • Job Families
    • Job Profiles
    • Job Levels
    • Role Groups
    • Role Levels
    • Role Group Skills
    • Role Level Skills
  • Detailed View: This tab presents the full imported data in a filterable table, allowing for a granular review.

Next Steps After Import

  • Review the processed data in the "Detailed View" tab for accuracy.
  • Link imported data to actual employee profiles within the system.
  • Utilize this comprehensive data for advanced skill gap analysis and to generate targeted training recommendations.

Export Job Architecture Data (Future Feature)

While currently a placeholder, future functionality will allow you to export your meticulously structured job architecture data (e.g., as a CSV file) for use in other HRIS or talent management systems, or for reporting purposes.
